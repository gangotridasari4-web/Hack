🌐 **NGO Donation Platform**

Hi! 👋
This is a full-stack project I built to make it easier for people to support causes they care about. The idea is simple — anyone can create a campaign, and others can donate to it.

💡 **Why I built this**

I wanted to understand how real-world platforms like donation apps work — handling users, campaigns, and tracking contributions.
This project helped me learn how frontend and backend actually connect in a practical way.

**What you can do here**

Sign up and log in 
Create your own fundraising campaign 
Donate to any campaign 
See how much money has been raised 
Track recent donations 

 **Tech I used**

Frontend
React
React Router
Axios
CSS

Backend
Node.js
Express
MongoDB (Mongoose)

📂** Project structure**
client/   → React frontend
server/   → Node + Express backend

**What I learned**
Connecting React with Express APIs
Handling authentication using JWT
Managing MongoDB data with Mongoose
Debugging real issues (which took most of the time 😄)
